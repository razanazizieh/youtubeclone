# YouTube Clone - UI Project

A professional, responsive YouTube homepage clone built with modern **HTML** and **CSS**. This project focuses on replicating the complex layouts of YouTube, including the header, sidebar, and the fluid video grid.

## 🚀 Live Demo
You can view the live site here: [**Live Preview Link**]([https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/](https://razanazizieh.github.io/youtubeclone/))

## ✨ Features
* **Modern Header:** Includes a functional-looking search bar, voice search, and notification badges.
* **Responsive Video Grid:** A clean 3-column layout (adjustable via CSS Grid) that showcases video thumbnails, titles, and channel info.
* **Sidebar Navigation:** A fixed left sidebar with consistent iconography.
* **Custom Tooltips:** Hover effects for icons in the header.
* **Mobile-Ready Concept:** Designed with scalability in mind using flexible layouts.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for better accessibility.
* **CSS3:** Advanced usage of:
    * **CSS Grid:** For the main video gallery.
    * **Flexbox:** For header alignment and video info positioning.
    * **Positioning:** Fixed headers and sidebars.
    * **Google Fonts:** Using the 'Roboto' typeface for an authentic look.

## 📂 Project Structure
* `index.html` - The main entry point.
* `styles/` - Organized CSS files (`header.css`, `sidebar.css`, `video.css`, `general.css`).
* `icons/` & `thumbnails/` - Visual assets used in the project.
